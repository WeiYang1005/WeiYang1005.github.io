记录下在HEXO写博文的方式

发表于 2019-12-15
字数统计: 254 字 | 阅读时长 ≈ 1 分钟

（鉴于太久没使用过HEXO来写博客了，后续可能也会忘掉，我就写一篇备忘博客来记录下吧！）

HEXO基础设置
-
Hexo 默认以标题作为文件名称，可以通过 Hexo 的配置文件 _config.yml 中的 new_post_name 参数来改变默认的文件名称，

new_post_name: :year-:month-:day-:title.md

例如：2019-12-15-title.md

同样在 Hexo 的配置文件 _config.yml 中, 配置 permalink 属性 ，来更改浏览器中访问的样式，

permalink: /blog/:year/:month/:day/:title/

例如：http://localhost:4000/blog/2019/12/15/hexo-blog/

创建一篇博客
-
（先要定位到hexo根目录）在命令行中输入

$ hexo new [layout] [title]

命令创建一篇新的文章，[layout]可以选择post(默认，即直接发布）、page（在source下新建一个文件夹）、draft（新建文件保存到_drafts文件夹中），也可以通过publish命令将_draft草稿移动到_posts发布文件夹下。

上传博客
-

hexo s -g #生成并本地预览

hexo d -g #生成并上传
