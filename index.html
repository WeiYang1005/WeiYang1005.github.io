重新搞下Hexo 0.0

发表于 2019-02-04

因为之前用的那个域名明天早上就到期了，准备换上前段时间腾讯云打折时候买的新域名，就上来改改看看。

很久没用hexo了，都忘记之前怎么设置的了= =然后又重新去看了下设置域名的方式，因为本身心情很乱，弄了一上午也没想明白怎么就更新不了CName里面的域名设定（真是傻得可以了）。然后看_config.yml里面突然发现一个叫做hexo-generator-cname的插件???? 才突然想起来原来我之前用插件锁定了CName里面的域名= =防止github自动deploy的时候出问题，然而居然把自己给关了。这就很尴尬了。。。。赶紧改掉，然后就有了现在这个地址了。

然后在改域名的时候在知乎看见一个很有意思的小东西，hexo里面会动的萌妹子插件，作为一个废宅 ，那肯定是要弄一个的呀，具体配置在github下面也相关说明，无非就是下载插件和live2d资源文件的过程了，大概就是

###
npm install –save hexo-helper-live2d
npm install {packagename}

###

packagename即要使用的live2d的package包名，譬如live2d-widget-model-miku

演示里面给出的效果好像是这样的，

,不过在评论里有看见别人的博客有自定义的live2d模型，后续可以了解下惹。

顺便也是有看了下md的语法和一些博客相关的设置，还是很有意思的，比如字数统计、阅读时间什么的，感觉需要学的东西还是很多的啦。不过也挺有意思的，算是学到了很多东西吧（雾，看书不好吗！
